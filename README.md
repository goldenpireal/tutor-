*Before Reading: This project is being built from scratch while I learn software engineering. The repository documents both the evolution of the software and my growth as a developer.*

\The project (*Non Technical part*)\
This is an open source software that helps with AI powered learning. Built from SCRATCH (this has been also study of software development for me)

**So these are the reasons why I created this software:** 

1) AI is soo bad at remembering, so this software stores your study sessions with it and then makes the AI remember who you are

2) I am very intereseted in programming, so I decided to build something useful while learning software engnineering 

3) And most of the good serivces are paid or works on premeium. It might very hard for a student to afford those subscriptions .. so I decided to build a software for free. By a Student for a Student

4) This also helps in self reflection of our progress
